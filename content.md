## Email
- List Order API related emails thru WorkIQ CLI

> workiq ask -q "List Orders API related emails. Show me its content"

## Microsoft Team Channel
-  List Orders API related messages thru WorkIQ CLI, from microsoft team channel `Order Service teams`

>workiq ask -q "List messages in Orders API teams. Show me its content"

## Both Microsoft Team Channel and Email
- List Orders API related messages and emails thru WorkIQ CLI, from microsoft team channel `Order Service teams` and email that contains `Orders API` in subject or body

>workiq ask -q "List messages in Orders API teams and emails. The content must be related to Orders API. Show me its content"
